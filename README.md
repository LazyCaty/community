### 校园社区系统
基于create-react-app构建 ,一个简易的校园社区系统，有常见的增删改查和后台管理功能  


### 使用

```
$ git clone https://github.com/LazyCaty/community.git
npm install
npm start
```

### 后台接口
[链接](https://github.com/LazyCaty/communityServer)

```
$ git https://github.com/LazyCaty/communityServer.git
npm install
npm start
```
